Informações sobre a utilização de banco de dados na aplicação Prepara!Concursos

Foi utilizado como teste o banco de dados PostgreSQL 9.6
Como driver JDBC de conexão ao banco foi utilizado o postgresql-42.1.4

Informações gerais:
   - nome do banco: prepara
   - usuário: postgres
   - senha: postgres

Backup do banco - "prepara.sql"
   - formato: Custom
   - seções: Pre-data, Post-data e Data
   - queries: Use Insert Commands, Include CREATE DATABASE statement, Include DROP DATABESE statement

Utilize a opção de "Restore" preparar o banco para trabalhar com ele
